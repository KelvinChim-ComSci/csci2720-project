CSCI2720 Project

Here is our site URL: http://csci2720-g96.cse.cuhk.edu.hk/

Before running the code, remember to install different node package, you can type the following commands:

$ npm init
$ npm install bcryptjs
$ npm install express
$ npm install cors
$ npm install mongoose

To start the Node server , you can type the following commands:

$ cd csci2720-project/src/Backend/

$ node index.js

Also, to start the npm, you can type the following commands:

$ cd csci2720-project/

$ npm start
